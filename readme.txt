*****************************************************************************
** ChibiOS/RT port for ARM-Cortex-M4 STM32F407.                            **
*****************************************************************************

** TARGET **

The demo runs on an ST STM32F4-Discovery board.

** The Demo **

The demo shows off different console functionality. A simple command shell is 
activated on virtual serial port SD2 via USB-CDC driver (use micro-USB plug on STM32F4-Discovery board).


** Build Procedure **

The demo has been tested by using the free GCC-based toolchain.
